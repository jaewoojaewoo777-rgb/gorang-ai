## [2026-06-20] 랜딩페이지·OAuth 쿠키·채널 연동창·영상 SIGKILL 수정 ✅

- 한 일:
  1. **서비스 소개 랜딩페이지** — 비로그인 시 `/`에 마케팅 소개 페이지 노출(히어로→핵심수치→기능3종→사용법3단계→요금제→하단CTA). 로그인 완료 사용자는 `/home` 자동 리다이렉트, 미등록은 `/register`
  2. **Google OAuth 세션 쿠키 누락 완전 수정** — Next.js 14 Route Handler에서 `getSession()`(cookies()) 및 `getIronSession(req,res)` 모두 redirect 응답에 Set-Cookie가 누락되던 문제를, `sealData()`로 직접 암호화 후 `response.cookies.set()`으로 박는 방식으로 해결. (이후 사용자/타 작업이 `COOKIE_DOMAIN` 옵션, no-store, 디버그 라우트 추가)
  3. **영상 페이지 BGM 패널 버그** — BGM 변경 버튼 클릭 시 `setModifyMode(null)` 제거 → 생성 중에도 패널 유지("생성 중..." 표시), 완료 후 자동 닫힘(handleGenerate 끝에서 setModifyMode(null))
  4. **채널 탭 연동 안내 창** — 틱톡/트립어드바이저/레드노트를 "심사중/준비중"→"연동하기"로 변경. connect 페이지에 `?flow=` 기반 GUIDES 가이드 3종 추가:

     - 트립어드바이저 → 비즈니스 관리 센터 안내 (공식 API 제한)
     - 레드노트 → 샤오홍슈 반자동 업로드 안내 (업로드 API 없음)
  5. **아이콘 정비** — 페이스북: 파란책📘→실제 페북 로고 SVG(파란 사각형+흰 f) / 레드노트: 📕→🇨🇳

  7. **자막 폰트 확대(6/19~20)** — 제목 ×4(120→480 세로), 부제목 ×4(80→320), 본문 ×2(64→128), 외국어 ×2(50→100). server.js PlayRes 4K 기준
